## Task Manager 

A full-stack **Task Manager application** built to manage daily tasks efficiently. The project combines a modern **React-based frontend** with a backend powered by **MongoDB**, focusing on clean architecture and scalable development practices.

##  Features
- Create, update, delete, and view tasks
- Interactive and responsive user interface
- REST API integration between frontend and backend
- MongoDB-based data persistence
- Easy to extend and maintain

##  Tech Stack

### Frontend

- CSS
